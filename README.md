# SenaiAdvocacia

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) na versão 19.0.1.

## Development server

Para rodar este projeto use o comando abaixo:

```bash
ng serve
```

Com o servidor up, acesse `http://localhost:4200/`. A aplicação recarrega automaticamente a cada atualização que fizer ao código.
